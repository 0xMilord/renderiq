# Renderiq SketchUp Plugin

Transform your SketchUp models into photorealistic renders using AI.

## Features

- **Camera Position Management**: Save, load, and manage camera positions
- **Screenshot Capture**: Capture current view at various resolutions
- **AI Rendering**: Generate photorealistic renders using Renderiq API
- **System Prompt Based**: No user prompt required - uses intelligent system prompts
- **Configurable Settings**: Quality, aspect ratio, style, and model selection

## Installation

### From Extension Warehouse (Coming Soon)

1. Open SketchUp
2. Go to **Window > Extension Warehouse**
3. Search for "Renderiq"
4. Click **Install**

### Manual Installation

1. Download the plugin RBZ file
2. Open SketchUp
3. Go to **Window > Preferences > Extensions**
4. Click **Install Extension**
5. Select the RBZ file
6. Restart SketchUp

## Setup

1. **Get API Key**
   - Sign up at [renderiq.io](https://renderiq.io)
   - Go to Dashboard > API Keys
   - Copy your API key

2. **Configure Plugin**
   - Open SketchUp
   - Go to **Extensions > Renderiq > Settings**
   - Enter your API key
   - (Optional) Enter default Project ID
   - Configure default render settings
   - Click **Save Settings**

## Usage

### Capture & Render

1. Position your camera in SketchUp
2. Go to **Extensions > Renderiq > Capture & Render**
3. Plugin captures current view
4. Render is sent to Renderiq API
5. Wait for render to complete
6. View and download rendered result

### Camera Management

**Save Camera Position:**
- Position camera
- Go to **Extensions > Renderiq > Save Camera Position**
- Enter a name
- Camera position is saved

**Load Camera Position:**
- Go to **Extensions > Renderiq > Load Camera Position**
- Select position from list
- Camera moves to saved position

**Manage Camera Positions:**
- Go to **Extensions > Renderiq > Manage Camera Positions**
- View all saved positions
- Load or delete positions
