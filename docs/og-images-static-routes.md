## File Naming Convention for Figma Export:

All OG images should be exported as:
- Format: JPG
- Dimensions: 1200x630px (standard OG image size)
- Location: `public/og/`
- Naming: Use the exact paths listed above (e.g., `about.jpg`, `contact.jpg`, `use-cases-concept-renders.jpg`)
