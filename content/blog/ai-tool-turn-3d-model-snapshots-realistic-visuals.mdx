## Step-by-Step: 3D Model to Render

### Step 1: Export Your Model View

From your 3D modeling software:

**SketchUp:**
1. Set up your view
2. Export as PNG or JPG
3. Use high resolution (1920x1080 minimum)
4. Include shadows if helpful

**Rhino:**
1. Set up your viewport
2. Export view as image
3. High resolution preferred
4. Clean background recommended

**Revit:**
1. Create a 3D view
2. Export as image
3. High quality settings
4. Include context if needed

### Step 2: Create a Project and Upload

1. Log into Renderiq (free tier available)
2. Navigate to `/render` or `/dashboard/projects`
3. Create a new project (e.g., "Office Building Visualization")
4. In the unified chat interface, upload your 3D model snapshot
   - Drag and drop or click to upload
   - Supported formats: JPG, PNG, WebP
