# Getting Started with Renderiq

Welcome to **Renderiq** - the AI-powered architectural visualization platform that transforms your sketches and prompts into photorealistic renders and videos using multiple state-of-the-art AI models.

## What is Renderiq?

Renderiq is a comprehensive AI architectural visualization platform designed for architects, designers, and creative professionals. Using multiple advanced AI models for image, video, and 3D generation, Renderiq enables you to:

- Transform architectural sketches into photorealistic visualizations
- Generate high-quality images and videos from text prompts
- Iterate on designs with version-controlled render chains
- Organize work into projects with advanced project management
- Share and discover renders in the public gallery

## Key Features

### AI-Powered Rendering
- **Image Generation**: Multiple state-of-the-art models for photorealistic renders
- **Video Generation**: Advanced video models for cinematic walkthroughs
- **3D Generation**: 3D model creation from images or text
- **Multiple Quality Levels**: Standard, High, and Ultra quality options
- **Aspect Ratio Control**: 16:9, 9:16, 1:1, 4:3, and 21:9 support

### Project Management
- **Project Organization**: Create and manage multiple architectural projects
- **Render Chains**: Version-controlled sequences of related renders
- **Automatic Chain Creation**: Default chains created automatically for new projects
- **Project Slugs**: URL-friendly project identifiers for easy sharing

### Unified Chat Interface
- **Conversational Rendering**: Chat-based interface for natural interaction
- **Image Upload Support**: Upload sketches and reference images
- **Version References**: Reference previous renders using @mentions (@v1, @v2, @latest)
- **Real-time Preview**: See renders as they're generated

### Canvas Editor
- **Node-Based Workflows**: Visual programming with React Flow
- **Workflow Automation**: Create complex rendering pipelines
- **Auto-save**: Automatic saving of canvas state

### Credit System
- **Flexible Billing**: Credit-based system for pay-as-you-go usage
- **Subscription Plans**: Monthly plans with included credits
- **Transaction History**: Track all credit usage and purchases

## Quick Start Guide

### Step 1: Create Your Account

1. Navigate to the [Sign Up page](/signup)
2. Choose your signup method:
   - **Email/Password**: Traditional signup with email verification
   - **OAuth**: Quick signup with Google or GitHub
3. Verify your email (if using email/password)
4. Complete your profile setup

### Step 2: Create Your First Project

1. Navigate to `/dashboard/projects` or `/render`
2. Click "New Project" or "Create Project"
3. Enter a project name and optional description
4. Upload an initial sketch (optional) or use the shape generator
5. Click "Create Project"

### Step 3: Generate Your First Render

1. Go to `/render` or select your project
2. Enter a descriptive prompt in the chat interface
3. Configure settings:
   - **Style**: Choose from predefined styles
   - **Quality**: Standard (1 credit), High (2 credits), or Ultra (3 credits)
   - **Aspect Ratio**: Select your preferred ratio
4. Click "Generate" or press Enter
5. Wait for your render to complete

### Step 4: Iterate and Refine

1. Review your generated render
2. Use version references to build on previous renders
3. Upload reference images for style transfer
4. Adjust prompts and settings for better results
5. Create render chains to track iterations

## System Architecture

Renderiq uses a modern, scalable architecture:

- **Frontend**: Next.js 15 with React 19 and TypeScript
- **Backend**: Next.js Server Actions for internal operations
- **API Routes**: External API endpoints for integrations
- **Database**: PostgreSQL with Drizzle ORM
- **Storage**: Supabase Storage for file management
- **Authentication**: Supabase Auth with OAuth support
- **AI Models**: Multiple models for image, video, and 3D generation

## Credit Costs

Understanding credit costs helps you plan your usage:

- **Image Generation**:
  - Standard Quality: 1 credit
  - High Quality: 2 credits
  - Ultra Quality: 3 credits

- **Video Generation**:
  - Base Cost: 5 credits
  - Duration Multiplier: Scales with video length
  - Model Multiplier: Higher quality models cost more

- **Upscaling**: Varies based on scale factor and quality
