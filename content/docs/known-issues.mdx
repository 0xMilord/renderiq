# Known Issues

This page documents known issues and their workarounds. We're constantly working to fix these issues.

## Authentication Issues

### Email Verification Not Received

**Issue**: Email verification email not arriving after signup.

**Solutions**:
- Check spam/junk folder
- Wait a few minutes (emails can be delayed)
- Request a new verification email
- Try a different email provider
- Contact support if issue persists

**Status**: Under investigation

### OAuth Login Fails

**Issue**: Google or GitHub login doesn't work.

**Solutions**:
- Clear browser cache and cookies
- Try a different browser
- Ensure you're using the same OAuth account you signed up with
- Check if OAuth account has proper permissions
- Try incognito/private mode

**Status**: Rare occurrence, usually browser-related

## Rendering Issues

### Slow Generation Times

**Issue**: Renders take longer than expected.

**Causes**:
- High quality settings (ultra quality is slower)
- Server load
- Complex prompts
- Large image uploads

**Solutions**:
- Use standard or high quality for faster results
- Simplify prompts
- Reduce image size
- Try again during off-peak hours

**Status**: Performance optimizations ongoing

### Poor Quality Results

**Issue**: Generated renders don't match expectations.

**Solutions**:
- Increase quality setting
- Add more detail to prompts
- Use reference images
- Try different styles
- Experiment with temperature settings
- See [Prompt Engineering](/docs/prompt-engineering) guide

**Status**: AI model improvements ongoing

### Generation Fails

**Issue**: Render generation fails with error.

**Common Causes**:
- Insufficient credits
- Invalid prompt
- Server error
- Network issues

**Solutions**:
- Check credit balance
- Verify prompt is valid
- Check internet connection
- Try again
- Contact support if issue persists

**Status**: Monitoring and improving error handling

## Image Upload Issues

### Upload Fails

**Issue**: Can't upload images.

**Solutions**:
- Check file format (JPG, PNG, WebP)
- Ensure file size under 10MB
- Try a different browser
- Clear browser cache
- Check internet connection

**Status**: File size limits may be adjusted

### Image Not Displaying

**Issue**: Uploaded image doesn't appear.

**Solutions**:
- Refresh the page
- Re-upload the image
- Check file isn't corrupted
- Try a different image
- Clear browser cache

**Status**: Under investigation

## Reporting Issues

### How to Report

If you encounter an issue not listed here:

1. **Check Status**: Verify it's not a known issue
2. **Gather Information**:
   - Browser and version
   - Device and OS
   - Steps to reproduce
   - Screenshots if applicable
   - Error messages
3. **Contact Support**: 
   - Email: support@renderiq.io
   - Include all gathered information

### What We Need

When reporting issues, please include:
- **Description**: What happened
- **Expected**: What should have happened
- **Steps**: How to reproduce
- **Environment**: Browser, OS, device
- **Screenshots**: If applicable
- **Error Messages**: Full error text
