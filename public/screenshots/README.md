# Screenshots Directory

This directory contains PWA screenshots for installation prompts and app stores.

## Required Files

### Desktop Screenshot
- **File:** `desktop-1.png`
- **Size:** 1280x720 pixels
- **Aspect Ratio:** 16:9
- **Purpose:** Desktop/wide form factor screenshot
- **Content:** Should showcase main dashboard or key features

### Mobile Screenshot
- **File:** `mobile-1.png`
- **Size:** 750x1334 pixels
- **Aspect Ratio:** 9:16 (portrait)
- **Purpose:** Mobile/narrow form factor screenshot
- **Content:** Should showcase mobile-optimized interface

## Generation Instructions

1. Take a screenshot of your desktop view at 1280x720
2. Take a screenshot of your mobile view at 750x1334
3. Save as PNG format
4. Optimize file sizes while maintaining quality
