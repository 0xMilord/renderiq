## File Format Recommendations

- **Preferred**: SVG format (scalable, best quality)
- **Alternative**: PNG format with transparent background
- **Size**: Recommended width: 120-200px, height: 60-80px
- **Background**: Transparent background preferred
- **Color**: Original brand colors (will be displayed with grayscale filter that removes on hover)

## Logo Sources

You can find official logos from:
- Company websites (usually in press/media kits)
- Brand asset repositories
- Official brand guidelines

## Notes

- Logos will be displayed in grayscale by default (60% opacity)
- On hover, logos become full color and 100% opacity
- Logos are displayed in a responsive grid (2 columns on mobile, 5 columns on desktop)
